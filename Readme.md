# Личный проект «Кексобукинг»

Кексобукинг — сервис размещения объявлений о сдаче в аренду недвижимости в центре Токио.
Пользователям предоставляется возможность размещать объявления о своей недвижимости
или просматривать уже размещённые объявления.

Репозиторий создан для обучения на интенсивном онлайн‑курсе
«[JavaScript. Профессиональная разработка веб-интерфейсов](https://htmlacademy.ru/intensive/javascript)»
от [HTML Academy](https://htmlacademy.ru).
